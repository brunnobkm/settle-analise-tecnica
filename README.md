# Settle · Análise Técnica (protótipo)

Protótipo de demonstração da nova **Análise Técnica** da plataforma Settle —
transforma a tela de informacional em orientada à decisão.

- Grid de itens type-agnostic (status + texto claro + resposta da análise)
- Tela cheia por item com a mecânica certa: comparação de produtos (matriz com
  ranking/score/recomendação), serviço/software (checklist atende/não) ou misto (seções)
- Resumo do Termo de Referência, origem da extração, confiança da IA, edição inline,
  congelar/redimensionar coluna, compartilhar/exportar

Protótipo estático (HTML/CSS/JS), sem build. Abra o `index.html`.
